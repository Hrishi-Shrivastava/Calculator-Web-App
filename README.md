# GUI Calculator

## Description
A fully functional, responsive Graphical User Interface (GUI) Calculator developed using HTML5, CSS3, and JavaScript (ES6). This project serves as a practical demonstration of fundamental web development principles, encompassing DOM manipulation, state management, and modern CSS layout techniques.

## Features
* **Core Arithmetic Operations:** Supports addition, subtraction, multiplication, division, and percentage calculations.
* **Input Validation:** Prevents invalid expressions and strictly restricts inputs to a single decimal point per operand.
* **Error Handling:** Safely manages edge cases, including displaying an error state upon division by zero.
* **Responsive User Interface:** Utilizes CSS Grid and Flexbox to maintain structural integrity across different screen sizes.
* **State Management:** Operates using custom JavaScript logic rather than relying on the `eval()` function, ensuring better security and operational control.

## Technologies Used
* **HTML5:** Semantic document structure.
* **CSS3:** Styling, layout configuration, and interactive state transitions.
* **JavaScript (ES6):** Core calculation logic and event handling.
